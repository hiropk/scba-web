# scba-web

## プロジェクト概要
島根県遊漁船組合の公式ホームページ。GitHub Pagesでデプロイする静的サイト。

## 使用技術

### フロントエンド
- HTML5（セマンティックマークアップ）
- CSS3（モダンなスタイリング、レスポンシブデザイン）
- JavaScript（Vanilla JS、軽量なインタラクティブ機能）
- 必要に応じて軽量なライブラリ（例: スライダー、モーダルなど）

### デプロイ
- GitHub Pages（静的サイトホスティング）

## サイト構成

### ページ構成
1. **トップページ（index.html）**
   - ヒーローセクション（メインビジュアル）
   - 組合の紹介（概要）
   - お知らせ・ニュース（最新3件程度）
   - 連絡先情報（簡易版）

2. **組合について（about.html）**
   - 組合の目的・活動内容
   - 役員紹介

3. **遊漁船情報（boats.html）**
   - 登録遊漁船一覧

4. **お知らせ（news.html）**
   - 最新情報一覧
   - お知らせ詳細ページ

### 共通コンポーネント
- ヘッダー（ナビゲーションメニュー、ロゴ）
- フッター（連絡先、コピーライト）

## デザイン方針

### カラーパレット
- メインカラー: 海をイメージしたブルー系
- アクセントカラー: オレンジ系（漁船・安全をイメージ）
- 背景色: 白・ライトグレー
- テキスト: ダークグレー・黒

### タイポグラフィ
- 見出し: ゴシック体（太め）
- 本文: ゴシック体（読みやすいウェイト）
- 日本語フォント: 游ゴシック、ヒラギノ角ゴなど

### レイアウト
- **スマートフォンファーストデザイン**（主要ターゲット）
  - モバイル画面（320px〜）を基準に設計
  - タッチ操作を考慮したUI（ボタンサイズ、間隔）
  - 縦スクロール中心のレイアウト
  - シンプルで直感的なナビゲーション
- **PC対応**（簡易対応）
  - タブレット以上（768px〜）で基本レイアウトを調整
  - 最大幅: 1000px程度（コンテンツを中央配置）
  - 2カラムレイアウトの検討（必要に応じて）
- カード型レイアウト（遊漁船情報、お知らせなど）

## 機能要件

### 必須機能
- モバイルナビゲーション（ハンバーガーメニュー、スライドメニュー）
- スムーススクロール
- 画像スライダー（トップページのヒーローセクション、タッチスワイプ対応）
- タップ・クリック操作の最適化

### 推奨機能
- 画像の遅延読み込み（Lazy Loading、モバイルでのパフォーマンス向上）
- お知らせの一覧表示（ページネーションまたは「もっと見る」ボタン）
- 電話番号のタップで発信（tel:リンク）
- アクセシビリティ対応（ARIA属性、適切なフォーカス管理）

## パフォーマンス最適化

- **モバイル最適化を優先**
  - 画像の最適化（WebP形式の使用、適切なサイズ、モバイル向けに圧縮）
  - CSS/JavaScriptの最小化
  - 不要なライブラリの排除（軽量な実装を優先）
  - 初回読み込み速度の最適化
- キャッシュ戦略の検討

## アクセシビリティ

- セマンティックHTMLの使用
- 適切な見出し階層（h1-h6）
- alt属性の適切な設定
- コントラスト比の確保
- キーボード操作の対応

## ブラウザ対応

- **モバイルブラウザを優先**
  - iOS Safari（最新2バージョン）
  - Chrome Mobile（Android、最新2バージョン）
- PCブラウザ（簡易対応）
  - Chrome、Firefox、Safari、Edge（最新バージョン）

## デザインの詳細方針

### モバイルUI/UX
- タップ領域: 最小44px×44px（指で操作しやすいサイズ）
- フォントサイズ: 本文14px以上、見出しは適切な階層
- 余白: 十分な余白を確保（コンテンツの可読性向上）
- スクロール: スムーズなスクロール体験
- ローディング: 軽量で高速な表示

### PC対応の考え方
- モバイルデザインをベースに拡張
- 最大幅を設定してコンテンツを中央配置
- 必要に応じて2カラムレイアウトを検討
- ホバー効果の追加（PCでの操作性向上）

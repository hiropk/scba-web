島根県遊漁船組合 規約
第1章　総則

第1条（名称）
本組合は、島根県遊漁船組合（以下「本組合」という）と称する。

第2条（所在地）
本組合の主たる事務所は、島根県内に置く。

第3条（目的）
本組合は、島根県内における遊漁船業者が相互に連携し、遊漁船業の健全な発展および組合員の利益の増進を図るとともに、地域漁業の振興並びに遊漁者の安全確保に寄与することを目的とする。

第4条（事業）
本組合は、前条の目的を達成するため、次の事業を行う。
1. 安全対策の検討および情報共有
2. 組合員相互の連絡および協力体制の構築
3. 地域漁業の振興および漁業関係者との連携に関する取組
4. 関係行政機関等への意見交換および連絡調整
5. その他、本組合の目的達成に必要な事業





第2章　組合員

第5条（組合員の資格）
本組合の組合員は、島根県知事による登録を受けて遊漁船業を営む事業者を対象とし、その代表者に限る。また、組合員は島根県内に居住する者でなければならない。

第5条の2（反社会的勢力の排除）
本組合は、暴力団、暴力団員、暴力団準構成員、暴力団関係企業その他これらに準ずる反社会的勢力を、組合員として受け入れず、また一切の関係を有しないものとする。

第6条（加入）
本組合に加入を希望する者は、島根県内に居住していることを証明する書類、並びに島根県内において遊漁船業を営んでいることを確認できる書類及び所定の加入申込書を提出しなければならない。
組合員の加入は、第16条（議決）の定めに従い、総会の議決をもって決定する。

第7条（脱会）
組合員は、組合長に届け出ることにより、脱退することができる。
除名手続中の組合員については、当該手続が終了するまで、脱退の効力を生じない。

第8条（除名）
組合員が次の各号のいずれかに該当する場合、総会の議決をもって除名することができる。
1. 本規約に著しく違反したとき
2. 組合員の資格要件を満たさなくなったとき
3. 本組合の名誉または信用を著しく損なう行為があったとき

第3章　役員等

第9条（役員の種別および定数）
本組合に、次の役員を置く。
組合長　1名
副組合長　1名
理事　2名

第10条（役員の選任および任期）
役員は、組合員の中から総会において選任する。
役員の任期は1年とし、再任を妨げない。

第11条（組合長および副組合長の職務）
組合長は、本組合を代表し、会務を総括する。
副組合長は、組合長を補佐し、組合長に事故あるときはその職務を代行する。

第12条（理事の職務）
理事は、組合長および副組合長を補佐し、本組合の運営に関する実務を分担して行う。
理事のうち1名を会計担当理事とし、本組合の会計事務を担当させることができる。






第4章　顧問

第13条（顧問）
本組合は、必要に応じて、1名又は複数名の顧問を置くことができる。
顧問は、組合長の要請に応じて助言を行うものとし、本組合の意思決定および業務執行には関与しない。
顧問は無報酬とする。

第5章　会議

第14条（総会）
総会は、本組合の最高意思決定機関とし、組合員をもって構成する。
総会の議決権を有するのは、組合員に限る。
総会は、原則として年1回開催する定時総会および必要に応じて開催する臨時総会とする。
総会は、組合長が招集する。

第15条（理事会）
理事会は、組合長、副組合長および理事をもって構成する。
理事会は、本組合の運営に関する事項を協議する。
理事会は、総会の権限に属しない事項について決定する。





第16条（議決）
会議の議決は、本規約に別段の定めがある場合を除き、出席者の3分の2以上の賛成をもって決する。
次の事項については、出席者全員の賛成をもって決する。
1. 規約の変更
2. 組合員の加入
3. 組合員の除名
4. 組合の解散

第6章　会計

第17条（会計年度）
本組合の会計年度は、毎年4月1日に始まり、翌年3月31日に終わる。

第18条（会費）
組合員は、総会の議決により定める会費を納入しなければならない。
既納の会費は、理由のいかんを問わず返還しない。

第19条（会計処理）
本組合の会計は、会計年度ごとに収支を明確にし、総会の承認を受けるものとする。

第7章　規約の変更および解散

第20条（規約の変更）
本規約の変更は、第16条（議決）第2項第1号の定めに従い、総会の議決により行うものとする。
第21条（解散）
本組合は、第16条（議決）第2項第4号の定めに従い、総会の議決により解散することができる。

附則
本規約は、令和7年12月9日より施行する。
